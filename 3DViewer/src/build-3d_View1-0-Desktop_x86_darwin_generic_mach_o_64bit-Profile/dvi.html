3D_Viewer

О программе

В данном проекте мы реализовали программу для просмотра 3D моделей в каркасном виде (3D Viewer). Модели загружаются из формата .obj и иметcя возможности: просматривать, вращать, масштабировать, перемещать объект,менять цвета(фона, линий, точек), менять размеры и выбирать вид( линий, точек).

Установка

Для того, чтобы начать использовать калькулятор вам нужно использовать команду make install.

Удаление

Для удаления программы вы можете использовать команду make uninstall.
